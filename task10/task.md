Типизация вложенного объекта с массивом
Условие: У тебя есть объект user, содержащий имя, возраст и массив друзей, где каждый друг описывается объектом с именем и возрастом.

Пример объекта:

```
   const user = {
    name: "Alice",
    age: 30,
    friends: [
        { name: "Bob", age: 25 },
        { name: "Charlie", age: 28 }
    ]
};
 
```
